# REQuest - ACKnowledge

Tool to construct, transform and analyze digital circuits based on elastic transactional protocol and Request-Acknowledge handshake.

## Use

### Node.js

```sh
npm i reqack --save
```

```js
const reqack = require('reqack');
```
